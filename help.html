# Настройка расширения

Для работы расширения необходимо создать в вк Standalone-приложение.

1. Переходим по ссылке - https://vk.com/editapp?act=create вводим название своего приложения, обязательно выбираем Standalone-приложение и кликаем Подключить приложение

2. После создания приложения, переходим в его Настройки:

3. Копируем ID приложения и вставляем в настройках расширения в поле APPID

4. Сохраняем настройки расширения.

# Добавление стикеров

Чтобы добавить новый стикер нужно кликнуть ПКМ по стикеру и выбрать Добавить стикер

! Требования к стикеру: формат *.png

! Если Вы добавляете стикер в первый раз, нужно будет разрешить доступ своему приложению

# Ресайз стиков

В настройках расширения можно включить ресайз стиков (изменение размера стикеров)
Для этого используется сервис cloudimage.io

Для начала нужно получить токен.

1. Регистрируемся по ссылке - https://www.cloudimage.io/en/register_page

2. После регистрации можно не вводить никаких данных, просто жмем Next >

3. В левом меню кликаем на Plan

4. Выбираем Annual billing и кликаем кнопку Get it

5. Заполняем поля CITY и COUNTRY (остальные поля можно не заполнять) и кликаем Save

6. Переходим на главную, копируем TOKEN и вставляем в настройках расширения:

7. Также в настройках указываем размер стикеров в пикселях (дефолт размер стикеров вк 128px)

! Настройка применяется только для новых загруженных стикеров
